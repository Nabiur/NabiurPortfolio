## Overview
This repository contains the source code for Nabiur's personal portfolio website. The site presents profile information, skills, featured projects, resume highlights, and contact details with a clean multi-page layout.

## Live site
GitHub Pages is enabled for this repository. If pages are configured to deploy from the default branch, the site can be accessed at:

`https://nabiur.github.io/NabiurPortfolio/`

## Pages included
- `index.html`: landing page and quick profile summary.
- `about.html`: background, strengths, and skill profile.
- `projects.html`: featured analytics and data projects.
- `resume.html`: professional experience and education highlights.
- `contact.html`: collaboration and contact information.

## Project structure
- `css/`, `scss/`: styling files.
- `js/`, `lib/`: scripts and front-end dependencies.
- `images/`, `fonts/`: media and typography assets.

## Tech stack
- HTML5
- CSS3 / SCSS
- JavaScript
- Bootstrap-based front-end components

## Run locally
1. Clone this repository.
2. Open `index.html` directly in your browser, or run a local static server:
   ```bash
   python -m http.server 8000
   ```
3. Visit `http://localhost:8000`.

## Customization ideas
- Add project filtering by category or tech stack.
- Add a downloadable PDF resume button.
- Add a contact form backend (for example Formspree or serverless function).
